AppyCrew OCR Widget – Demo

Scan forms with AppyCrew OCR

This page is just for testing and documentation. In production you only need a single <script> tag.

Demo form

Fill this form automatically by clicking the floating “Scan with AppyCrew” button in the corner.

How to use this widget in your app

  1. Push this folder to a new GitHub repo, for example
    appycrew-ocr-widget.
  2. Connect the repo to Vercel (Framework: “Other”).
    Leave the build command empty – Vercel will serve index.html and the /api folder automatically.
  3. In Vercel Project Settings → Environment Variables, add your OCR.space API key:
    OCR_SPACE_API_KEY = your-key-here
  4. After deploy, your widget URL will look like:
    https://YOUR-PROJECT.vercel.app/ocr-widget.js
  5. On any form page (on any site), add this snippet: 
    <script src="https://YOUR-PROJECT.vercel.app/ocr-widget.js" defer></script>

The widget only shows when a <form> is present on the page.
